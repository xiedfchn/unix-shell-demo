# unix shell demo
A Shell provides you with an interface to the Unix system.

## Veriable
`NAME=Michael`\
`echo $Name`

## Special Variables
$0 is the name of the file\
$1 is the first parameter\
$2 is the second parameter

## Array
`NAME[0]="Zara"`\
`NAME[1]="Qadir"`\
`echo "First Index: ${NAME[0]}"`
